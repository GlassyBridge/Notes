The Web

The Web (or World Wide Web) is a collection of documents and other files. Most of the documents are made using Hypertext Markup Language (HTML).

What is HTML?

HTML: stands for Hyper Text Markup Language.

HTTP(S): stands for Hyper Text Transfer Protocol (S - refers to secure).
---------It is the protocol that tells the server what the browser is sending it.

Markup

Markup: refers to a text that has a special meaning.

  • opening tags use greater and less than symbols.
  • closing tags use / with those symbols.

Elements: are tags that are complete/ have opening and closing tags.
______Example: p, li, ul, ol, h1...6, em/i, strong/b, div
_________Some of these don't require closing tags. (p, li)
Empty (void) elements: are elements that don't have closing tags.
______Example: br, img
